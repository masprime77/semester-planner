Lectio GitHub ↗

Plan your semester, calmly.

A minimal desktop app for tracking courses, weekly readings, and tasks — with click-to-cycle status badges and per-course progress. No account, no server: every semester is a plain file on your machine.

What it does

Week & course views

See everything by week, or switch to per-course columns. Whatever fits how you study.

Click-to-cycle status

Readings cycle pending → seen → summarized → studied. Tasks cycle not done → done → reviewed.

Progress at a glance

Per-course progress bars and a current-week indicator keep the whole semester in view.

Your files, offline

No account and no server. Each semester is a plain JSON file saved locally on your machine.

Light, dark & auto

A clean, minimal interface that follows your system theme — or pick light or dark yourself.

Auto-updates

New versions download and install themselves, so you're always on the latest release.
